# Database Schema Guide

This document outlines the core tables used in the InsightFlow database.

## System Tables
- **Users**: User credentials, names, and superuser status.
- **Datasets**: Ingested file metadata, column schemas, statistics, and table mapping.
- **Dashboards**: Configurations, titles, layouts, and filters.
- **Charts**: Metric mappings, coordinates, and configurations.
- **QueryHistories**: Track user SQL queries and translation history.
