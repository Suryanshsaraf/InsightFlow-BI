# API Endpoint Reference

This guide provides a list of REST endpoints exposed by the InsightFlow API.

## Endpoints
- `POST /api/v1/auth/signup`: User registration
- `POST /api/v1/auth/login`: User session creation
- `POST /api/v1/datasets/upload`: Upload dataset CSV file
- `GET /api/v1/datasets/`: List datasets
- `GET /api/v1/dashboards/`: List dashboards
